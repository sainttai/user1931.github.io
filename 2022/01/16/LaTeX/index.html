Adding images

Use package graphicx to add images.

example:

The command \graphicspath{{img/}} tells \(\LaTeX{}\) that the images are kept in a folder named img under the current directory.

The \includegraphics{universe} command is the one that actually included the image in the document. Here universe is the name of the file containing the image without the extension, then universe.PNG becomes universe. The file name of the image should not contain white spaces nor multiple dots.

Abstract & Newlines and paragraph

note:

1.When writing the contents of your document, if you need to start a new paragraph you must hit the "Enter" key twice (to insert a double blank line). Notice that \(\LaTeX{}\) automatically indents paragraphs.

2.To start a new line without actually starting a new paragraph insert a break line point, this can be done by \\ (a double backslash as in the example) or the \newline command.

Chapters and sections

Commands to organize a document vary depending on the document type, the simplest form of organization is the sectioning, available in all formats.

-1 \part{part}
0 \chapter{chapter}
1 \section{section}
2 \subsection{subsection}
3 \subsubsection{subsubsection}
4 \paragraph{paragraph}
5 \subparagraph{subparagraph}

note: The command \part and \chapter are only available in report and book document classes.
